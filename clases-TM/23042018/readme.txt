El objetivo del ejercicio es validar un formulario simple utilizando jQuery.
Campos obligatorios
Se incorpora funcionalidad de borrar formulario y limpiar mensajes de error.

El archivo validarEach.js implementa el iterador propio de jQuery. 100 % funcional.

El archivo validar, contiene una comparación de versiones de código. Each y forEach, se deben comentar las versiones 1 o 2 para probar cada caso.
